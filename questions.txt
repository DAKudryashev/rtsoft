- Разница выдается отсортированной по ФБ.
- Имеем дело со стандартной процедурой создания проекта: создается один Application, создается приложение, назначается
вычислительный ресурс, сохраняется; блоки без application не добавляются. Учитываем, что в файле sys: один Application,
один SubAppNetwork, один Device, МИНИМУМ один Resource, на каждом из них один FBNetwork
